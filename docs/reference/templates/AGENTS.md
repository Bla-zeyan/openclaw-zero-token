# AGENTS.md

This is your OpenClaw agent workspace configuration file.

## About

This file defines the agent's personality, behavior, and capabilities for this workspace.

## Usage

Edit this file to customize how your agent behaves. You can:

- Define the agent's role and personality
- Set behavioral guidelines
- Configure tool permissions
- Add custom instructions

## Example

```markdown
# Agent Configuration

## Role
You are a helpful assistant that helps users with their tasks.

## Behavior
- Be concise and helpful
- Ask clarifying questions when needed
- Provide code examples when appropriate

## Tools
- filesystem: read, write
- shell: execute commands
```

For more information, see the OpenClaw documentation.
